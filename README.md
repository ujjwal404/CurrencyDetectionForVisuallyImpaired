# Indian-Currency-Recognition
A project for visually impaired people to recognise the bills of Indian National Rupee using Deep Learning technology.

### Dataset:
```
currency\
  ten\
  twenty\
  fifty\
  hundrad\
  fivehundrad\
  thousand\
```
